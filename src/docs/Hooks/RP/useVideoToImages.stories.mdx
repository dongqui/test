# useVideoToImages

비디오에서 컷 이미지를 추출하는 hook

#### example

```javascript
  useVideoToImages({ videoRef, action, active, intervalTime });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **videoRef(DOM)**        | video 태그에 있는 DOM ref값        |
| **action(Function)**        | 컷 이미지를 다 모았을때 넣고 싶은 액션        |
| **active(boolean)**        | 컷 이미지가 이미 모아져 있으면 활성화시킬필요가 없기 때문에 넣었음        |
| **intervalTime(number)**        | setInterval 간격        |
