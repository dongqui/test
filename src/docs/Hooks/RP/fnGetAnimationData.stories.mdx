# fnGetAnimationData

모델파일에 들어있는 animation data 를 가져오기 위한 hook

#### example

```javascript
  const { result, error, msg } = await fnGetAnimationData({ url });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **url(string)**        | 모델파일의 url        |

## Return

| Return | Description |
| ------ | ----------- |
| **result(motionTypes[] 참고)**        | 애니메이션 데이터 배열        |
