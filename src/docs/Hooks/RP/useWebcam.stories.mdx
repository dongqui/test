# useWebcam

웹캠의 촬영에 관여하는 hook

#### example

```javascript
  useWebcam({ videoRef, isRecording: recordingData.isRecording, onComplete, onError });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **videoRef(DOM)**        | video 태그에 있는 DOM ref값        |
| **isRecording(boolean)**        | 녹화중인지 여부        |
| **onComplete(Function)**        | 완료되었을때 액션        |
| **onError(Function)**        | 오류가 발생했을때 액션        |
