# useOutsideClick

특정 돔 바깥쪽을 클릭했을때 발생시키고 싶은 이벤트를 구현해주는 hook입니다.

#### example

```javascript
useOutsideClick({ ref, event: () => {} });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **ref(DOM)**        | 타겟이 되는 DOM        |
| **event(Function)** | 발생시키고 싶은 이벤트 |
