# useTimer

타이머를 켜서 시간을 리턴시켜주는 hook

#### example

```javascript
const { result } = useTimer({ isActive, totalTime });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **isActive(boolean)** | 활성화여부 |
| **totalTime(number)** | 총 소요시간 |

## Return

| Return | Description |
| ------ | ----------- |
| **result(number)** | 현재 초 |
