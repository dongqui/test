# useWindowResize

윈도우 창크기를 조절할때 발생시키고 싶은 이벤트를 넣는 hook

#### example

```javascript
  useWindowResize({
    event: ({ width, height }) => {
      setTpSize({ width, height: height * TIMELINEPANEL_INFO.heightRate });
    },
  });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **event({ width: number, height: number })** | width: 가로길이, height: 세로길이 |
