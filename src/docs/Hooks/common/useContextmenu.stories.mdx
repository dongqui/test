# useContextMenu

오른쪽 마우스를 클릭했을때 발생시키고 싶은 이벤트를 넣어주는 hook

#### example

```javascript
useContextMenu({ targetRef: iconViewWrapperRef, event: onContextMenu });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **targetRef(DOM)**        | 타겟이 되는 DOM        |
| **event(Function)** | 발생시키고 싶은 이벤트 |
