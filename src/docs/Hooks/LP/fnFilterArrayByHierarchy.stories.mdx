# fnFilterArrayByHierarchy

LP listview 에서 파일검색시 검색어로 필터링을 해주는 hook

#### example

```javascript
cosnt data = fnFilterArrayByHierarchy({ data, searchWord });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **data(mainDataTypes[])**        | 메인데이터        |
| **searchWord(string)**        | 검색어        |

## Return

| Return | Description |
| ------ | ----------- |
| **data(mainDataTypes[])**        | 정렬 후의 메인데이터        |
