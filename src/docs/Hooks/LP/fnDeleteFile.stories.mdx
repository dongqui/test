# fnDeleteFile

파일을 삭제시켜주는 hook. 그냥 해당 파일만 삭제하면 되는게 아니라 하위에 있는 모든것들을 삭제해야 되서 따로 hook을 만들었음

#### example

```javascript
fnDeleteFile({ mainData });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **mainData(mainDataTypes[])**        | 메인데이터        |
