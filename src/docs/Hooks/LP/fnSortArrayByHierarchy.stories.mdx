# fnSortArrayByHierarchy

data를 계층구조 순서대로 정렬시켜주는 hook

#### example

```javascript
const data = fnSortArrayByHierarchy({ data: filteredData });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **data(mainDataTypes[])**        | 메인데이터        |

## Return

| Return | Description |
| ------ | ----------- |
| **data(mainDataTypes[])**        | 정렬 후의 메인데이터        |
