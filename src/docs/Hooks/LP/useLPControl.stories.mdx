# useLPControl

LP에서 파일복사, 삭제, 이름바꾸기, 붙여넣기 등 여러가지 작업을 하기 위한 기능을 가진 함수들을 모아놓은 hook

#### example

```javascript
const {
    onClick,
    onContextMenu,
    onCopy,
    onDragStart,
    onDragStop,
    onDrop,
    onEdit,
    onPaste,
    shortcutData,
    filteredData,
  } = useLPControl({ contextmenuInfo, mainData, pages, searchWord });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **contextmenuInfo(contextmenuTypes)**        | 컨텍스트메뉴정보        |
| **mainData(mainDataTypes)**        | 메인데이터        |
| **pages(pagesTypes)**        | 페이지정보        |
| **searchWord(string)**        | 검색어        |

## Return

| Return | Description |
| ------ | ----------- |
| **onClick(Function)**        | 아이콘 클릭        |
| **onContextMenu(Function)** | 오른쪽메뉴클릭
| **onCopy(Function)** | 복사
| **onDragStart(Function)** | 드래그시작
| **onDragStop(Function)** | 드래그끝
| **onEdit(Function)** | 이름수정
| **onPaste(Function)** | 붙여넣기
| **shortcutData(Function)** | 단축키정보
| **filteredData(mainDataTypes)** | 검색어로 필터링을 한 메인데이터
