# fnGetFileName

파일이름에서 기존에 중복된 파일이름이 있을 경우 숫자를 붙여주는 hook

#### example

```javascript
fnGetFileName({ key, name, mainData });
```

## Props

| Prop                | Description            |
| ------------------- | ---------------------- |
| **key(string)**        | 키값        |
| **name(string)**        | 추가 또는 변경하고자 하는 파일 이름        |
| **mainData(MainDataTypes)**        | 메인데이터        |

## Return

| Return | Description |
| ------ | ----------- |
| **result(string)**        | 수정된 파일이름        |
